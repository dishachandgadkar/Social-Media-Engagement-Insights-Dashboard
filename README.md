# Social-Media-Engagement-Insights-Dashboard
Interactive Power BI dashboard analyzing social media engagement metrics, audience sentiment, platform performance, posting trends, best posting times, and geographical insights using DAX, Power Query, ETL processes, and data visualization techniques.

## 🚀 Project Overview
Developed an interactive Power BI dashboard to analyze social media engagement metrics, audience sentiment, platform performance, posting trends, and geographical insights. The project focuses on transforming raw social media data into meaningful business insights using Power BI, DAX, and Power Query.

---

## 📌 Key Features
- KPI Cards for Total Likes, Shares, Comments, Engagement & Impressions
- Platform-wise Engagement Analysis
- Audience Sentiment Analysis
- Monthly Engagement Trend Visualization
- Best Posting Time Analysis
- Geographical Engagement Mapping
- Interactive Slicers & Filters
- Professional Dashboard Design with Icons & Visuals

---

## 🛠 Tools & Technologies Used
- Power BI
- DAX (Data Analysis Expressions)
- Power Query Editor
- ETL Process
- Microsoft Excel / CSV Dataset
- Data Visualization Techniques

---

## 📈 Dashboard Insights
- Identified top-performing social media platforms
- Compared engagement trends across platforms
- Analyzed audience sentiment distribution
- Determined best posting hours for maximum engagement
- Visualized audience engagement geographically

---

## 📂 Dataset
Social Media Engagement Dataset containing:
- Platform Data
- Likes, Shares, Comments
- Engagement Metrics
- Sentiment Data
- Posting Time & Date
- Location-based Information

---

## 🌟 Outcome
Successfully created an interactive business intelligence dashboard that helps analyze social media performance and supports data-driven decision-making.
